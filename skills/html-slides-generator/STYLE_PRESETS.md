# 精选视觉预设库 — 37 个风格

每个预设包含：名称、氛围、配色、字体、签名元素、最适场景。Phase 2 从此库中精选 3 个推荐给用户。

## 暗色 / 科技 (Dark & Tech)

### 01. 暗黑科技未来 (Dark Tech Future)
**Vibe**: 沉浸式暗色、霓虹未来感、产品发布
**Colors**: `#0d0d0f` bg | `#00f0ff` cyan | `#7b3fff` violet | `#e0e0e0` text
**Fonts**: Space Grotesk / Inter / JetBrains Mono
**Signature**: 细霓虹描边发光、HUD 分隔线、微妙网格背景
**Best for**: 科技产品发布、AI/Web3 主题、开发者大会

### 02. 深夜编辑 (Late-Night Editorial)
**Vibe**: 黑画布 + 亮粉点缀 + 珍珠奶油文字，深夜杂志的柔软锐利
**Colors**: `#0a0a0a` black | `#ff3b8b` pink | `#fff5f0` pearl cream
**Fonts**: Instrument Serif italic / Inter
**Signature**: 亮粉细下划线、大段暗色留白、斜体标题
**Best for**: 时尚/奢侈品牌、创意作品集、高端 Keynote

### 03. 像素霓虹街机 (Pixel Arcade Neon)
**Vibe**: 深海军蓝虚空中的霓虹像素艺术
**Colors**: `#0a0e27` void | `#ff00ff` magenta | `#00ffff` cyan | `#ffff00` yellow
**Fonts**: Press Start 2P / Fira Code
**Signature**: 像素化网格、霓虹发光 text-shadow、扫描线、阶梯故障
**Best for**: 游戏/电竞、创意娱乐、青年文化

### 04. 高电压设计工作室 (High Voltage Studio)
**Vibe**: 黑色帆布 + 亮黄色字体，电力十足
**Colors**: `#000000` black | `#ffe600` yellow | `#ffffff` white
**Fonts**: Bebas Neue / Space Mono
**Signature**: 亮黄粗线 4px、超大无衬线标题 120px+、黑色负空间
**Best for**: 设计工作室、创意路演、独立品牌
