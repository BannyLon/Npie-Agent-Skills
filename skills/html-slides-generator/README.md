# HTML Slides 生成器 (HTML Slides Generator)

将你的想法、笔记、文档一键转化为**零依赖、单文件、极致美观的交互式网页幻灯片**。不需要任何前端知识，不需要安装任何软件——只要一个浏览器。

---

## 快速开始

### 触发技能

在 Claude Code 中说：

```
帮我做一个关于 XXX 的 slides
```

或者直接提供文件：

```
把这份文档转成网页 PPT
```

### 完整流程（3 步）

```
你说主题 / 上传文件
       ↓
Claude 输出内容大纲，你确认
       ↓
Claude 推荐 3 个视觉风格 + 生成视觉预览 HTML，你选 1 个
       ↓
Claude 生成单个 .html 文件 → 完成 ✅
```

> **新增**：Phase 2 推荐风格时会同时生成一份 `xxx.preview.html` 视觉预览文件，三张卡片并排展示配色、字体和微型幻灯片效果。打开即可直观对比，不再靠文字脑补。

### 示例

```
用户：帮我做一个关于 AutoFlow 2.0 产品发布的 slides

Claude：你有相关的文字资料吗？（如果没有，我基于已知信息继续）

用户：没有

Claude：
  ## 建议结构（12 页）
  1. 封面 — AutoFlow 2.0
  2. 目录 — 概览 / 性能 / 新能力 / 定价
  ...确认？

用户：确认，演讲辅助，暗黑风格

Claude：
  ### 推荐 A：暗黑科技未来 — 青霓虹 + Space Grotesk
  ### 推荐 B：深夜编辑 — 亮粉 + Instrument Serif
  ### 推荐 C：安静学术蓝金 — 暖金 + Cormorant
  🖼 已生成 opus48.preview.html 视觉预览，打开查看三款风格的实际效果
  > 内嵌 37 个预设，不满意可换

用户：选 A

Claude：[输出完整 .html 文件] ✅
```

## 如何使用生成的 HTML 文件

### 基本操作

| 操作 | 方式 |
|------|------|
| **翻页** | 键盘 ← → 方向键、空格键 |
| **后退** | Shift + Space |
| **跳转首尾** | Home / End 键 |
| **鼠标滚轮** | 上下滚动翻页（自动防连跳） |
| **触屏滑动** | 左右或上下滑动 |
| **全屏** | 按 `F` 键进入全屏演示模式 |
| **退出全屏** | 按 `F` 或 `Esc` |

### 保存与分享

1. **保存**：将 `.html` 文件保存到任意位置，双击用浏览器打开
2. **发送**：直接发送 `.html` 文件给他人——单文件，对方双击即可观看
3. **导出 PDF**：浏览器打开 → `Ctrl+P`（Mac: `Cmd+P`）→ 另存为 PDF
4. **部署到线上**：
   - **Vercel**：运行 `bash scripts/deploy-vercel.sh <file.html>`
   - **GitHub Pages**：推送到 `gh-pages` 分支
   - **Netlify**：拖拽 `.html` 文件到 Netlify Drop

### 手动修改

每个生成的 HTML 文件都带有清晰注释，方便你自行微调：

- **修改配色**：搜索 `STYLE:` 定位 CSS 变量区
- **替换图片**：搜索 `IMG:` 定位图片占位位置
- **修改文字**：搜索幻灯片编号（如 `Slide 1`）定位内容区
- **调整字号**：搜索 `font-size` 在 CSS 变量区修改

## 其他功能

### PowerPoint 转换

如果你有 `.pptx` 文件，直接提供给 Claude，会自动提取文本和图片，用选定风格重新设计：

```
用户：[上传 presentation.pptx] 把这个转成网页 slides
```

### 持续迭代

生成后可以随时要求修改：

- "把第 3 页改成两列布局"
- "整体色调换成更暗的"
- "标题换成衬线体"
- "在总结页前加一页数据对比"

每次输出**修改后的完整新 HTML 文件**（不是 diff），你可以随时保存使用。

## 技术特性

- **零依赖**：单文件包含所有 CSS/JS，不需要 npm install、不需要框架
- **固定 16:9**：1920×1080 虚拟画布，自动缩放适配任意屏幕
- **完整交互**：键盘/鼠标/触屏全支持，进度条，页码显示
- **无障碍**：支持 `prefers-reduced-motion`，自动关闭动画
- **体积控制**：纯文本 ≤30KB，含图片 base64 ≤500KB
