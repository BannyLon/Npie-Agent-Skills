# 书的金字塔 & 精读判断树

## 书的金字塔

```
        ┌─────────────┐
        │ ＜100 本     │ ← 跟你一起成长的书。重读时发现书好像也成长了。
        │ 最高层       │   每次重读都能看到新东西。值得分析阅读多次。
        ├─────────────┤
        │ ＜几千本     │ ← 值得做一次分析阅读。读完后心智增长了，
        │ 中间层       │   但再读不会再获得更多。精华已被吸收。
        ├─────────────┤
        │ 99.9%        │ ← 消遣娱乐 / 获取资讯用。扫读即可。
        │ 底层         │   不值得分析阅读，也不值得重读。
        └─────────────┘
```

## 精读判断树

```
拿到一本书
    │
    ├── 这是小说/诗歌/戏剧吗？
    │     ├── 是 → 用虚构类读法（见 reading-by-type.md#想象文学）
    │     │       精读 ≠ 逐句分析，精读 = 沉浸体验 + 重读
    │     │
    │     └── 否 → 继续 ↓
    │
    ├── 检视阅读（15-30 分钟）
    │     ├── 看目录、序言、索引、出版者介绍
    │     ├── 挑几个章节翻翻
    │     ├── 翻翻结尾
    │     └── 结论 ↓
    │
    ├── 这本书在谈什么？我关心这个主题吗？
    │     ├── 不关心 → 放下。世界上有太多好书，不值得在不关心的事上花时间
    │     └── 关心 → 继续 ↓
    │
    ├── 这本书的架构清晰吗？作者知道自己在说什么吗？
    │     ├── 不清晰 → 可能是烂书。放下，或至少不做分析阅读
    │     └── 清晰 → 继续 ↓
    │
    ├── 粗浅阅读一遍（碰到不懂的不停、不查）
    │     └── 结论 ↓
    │
    ├── 我读懂了多少？
    │     ├── 50% 以下 → 可能超出当前能力。先放一放，积累后再来
    │     ├── 50-80% → 值得做分析阅读
    │     └── 80% 以上 → 可以进入分析阅读，也可以直接进入评论
    │
    └── 🎯 进入分析阅读（三个阶段递进）
          ├── 阶段一：结构（规则 1-4）
          ├── 阶段二：诠释（规则 5-8）
          └── 阶段三：评论（规则 9-15）
```

## 「值得精读」的判断标准

| 信号 | 含义 |
|------|------|
| 读完后你还在想它 | 值得重读 |
| 你想跟别人讨论它 | 值得分析阅读 |
| 你已经在书上做了大量标记 | 数据已就绪，可以分析阅读 |
| 你觉得懂了，但说不出一个完整的句子概括它 | 你还没懂，需要阶段一 |
| 你能概括，但不能用自己的话复述核心论点 | 需要阶段二 |
| 你能复述，但说不出它哪里对、哪里不对 | 需要阶段三 |
| 读完后你的某个看法改变了 | 这本书进入了你的生命 |
| 读完后你觉得「也就那样」 | 要么是烂书，要么你没读进去 |

## 培根的忠告

> 有些书可以浅尝即止，有些书是要生吞活剥，只有少数的书是要咀嚼与消化的。

分析阅读 = 咀嚼与消化。检视阅读 = 浅尝。主题阅读 = 将多本咀嚼过的书融会贯通。
